[ 모듈 폴더구성 정의 ] : components 폴더내부

1.  layout : 레이아웃 컴포넌트 
        영역별 layout
        (0) layout
        (1) top area
        (2) main area
        (3) footer area
2. pages : main area에 들어갈 메인 컴포넌트
        (1) Main : 첫 페이지
        (2) News : 뉴스 페이지
        (3) Board : 게시판 페이지
        ...

3. modules : page컴포넌트 내부 부분 컴포넌트
4. func : 기능구현 js 파일
5. data : data객체 파일
6. plugin : plugin모듈 컴포넌트

[기타 폴더 구성요소]
1. css : 스타일 시트 파일
2. static : 정적구성파일 / ex) 웹폰트...
